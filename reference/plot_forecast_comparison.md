# Compare Forecast Outputs

**\[deprecated\]**

This function is deprecated and may be removed in a future release.

This function visualizes and compares multiple forecast outputs
generated by
[`generate_forecast()`](https://bccdc-phsa.github.io/ViroReportR/reference/generate_forecast.md).
It plots median predicted incidences with 80% and 90% predictive
intervals.

## Usage

``` r
plot_forecast_comparison(...)
```

## Arguments

- ...:

  Named forecast result objects produced by
  [`generate_forecast()`](https://bccdc-phsa.github.io/ViroReportR/reference/generate_forecast.md).
  Each object should contain a data frame `forecast_res_quantiles` with
  columns:

  - `date`: Forecast date

  - `p10`, `p50`, `p90`: 10th, median, and 90th quantile predictions

  - (Optional) other quantiles if available

## Value

A **ggplot2** object showing forecasted median incidences over time with
shaded prediction intervals, colored by input source (i.e., model name).
